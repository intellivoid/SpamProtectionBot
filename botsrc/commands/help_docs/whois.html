{Whois Lookup Command} {This command will allow you to to lookup information about a channel, chat or user if it's available to this bot.} {Usage Examples:} /whois - {This will return information about yourself} /whois -c - {This will return information about the current chat} /whois [ID/PTID/{Username}/({Reply})] - {Returns information about the channel, chat or user} {Options:} -c / --chat - {Returns information about the current chat} -p / --private - {Returns the results privately} {What's the trust prediction?} {A trust prediction is the trust value of a user, the higher the better} {What does Potential Spammer mean?} {A general indication determined by the bot if the user is believed to be an potential spammer} {Who are users verified by Intellivoid Accounts?} {These are users who linked their Telegram Account to their Intellivoid Account} {What are verified/official channels and groups?} {These are chats or channels that were deem official by Intellivoid} {What is a private ID?} {It's a simple ID used to mask the real ID of a user or client}
