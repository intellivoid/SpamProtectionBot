Blacklist Command This command will allow operators to alter the blacklist flag of a user Usage: /blacklist (In reply to target user) -r [Blacklist Flag] /blacklist (In reply to forwarded content) -f -r [Blacklist Flag] /blacklist (Mention) -r [Blacklist Flag] /blacklist -u [Private Telegram ID] -r [Blacklist Flag] /blacklist -u [User ID] -r [Blacklist Flag] /blacklist -u [Username] -r [Blacklist Flag] Options: -f - Targets the original sender of the forwarded message -p, --private - Returns the results privately -u, --user, --target PTID/ID/Username - The specified target user -r, --reason, --flag Flag Name - The specified blacklist flag -o, --optid PTID - The original Private ID required for blacklisting a user with the flag 0xEVADE -s, --silent - Suppresses success responses only (Silent mode) -cs, --complete-silent - Suppresses all responses (Complete silent mode) --info - Returns basic information about the command How do a blacklist a user? Send /blacklist -u @ExampleUsername -r 0xSPAM How do I remove a blacklist? Use the flag 0x0, for eaxmaple /blacklist -u @ExampleUsername -r 0x0 How do I run this command silently? Suppress success messages only: /blacklist -s -u @ExampleUsername -r 0xSPAM Suppress all messages: /blacklist -cs -u @ExampleUsername -r 0xSPAM How do I log and blacklist at the same time? Include the -r command like /log -r 0xSPAM, send /help log for more details What are the available blacklist flags? Send /help blacklist_flags
