Prop Command This command will allow the main operator to modify properties of any chat, channel or user Usage: /prop --ptid [Private Telegram ID] --pointer [Value Pointer] --set-value [Set Value] Options: --ptid - The target Private Telegram ID --pointer - The value pointer --set-value - The value to set to the current pointer --info - Returns basic information about the command What are the pointer values for a user? Send /help prop_user What are the pointer values for a chat? Send /help prop_chat What are the pointer values for a channel? Send /help prop_channel
