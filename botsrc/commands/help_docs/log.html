Log Command This command will allow operators/agents to manually log spam or abuse as evidence Usage: /log (In reply to target user) /log (In reply to forwarded content) -f /log (In reply to forwarded content) -f -t Options: -f - Targets the original sender of the forwarded message -t - Used with the -f option, this will target the sender and the original sender -p, --private - Returns the results privately -s, --silent - Suppresses success responses only (Silent mode) -cs, --complete-silent - Suppresses all responses (Complete silent mode) --info - Returns basic information about the command Why can't the the bot get the forwarded user/channel? This is because of privacy settings or the entity was deleted How can I blacklist while logging? Provide the -r option followed by a blacklist reason, for example /log -r 0xSPAM -f How do I run this command silently? Suppress success messages only: /log -s Suppress all messages: /log -cs
