SpamProtectionBot SpamProtectionBot allows you to protect your group from spam, unwanted users and potential spammers. When you add this bot to your group, make sure that it has the following permissions in order to operate correctly. - Delete Messages - Ban Users The /help command can provide a lot of useful information, send the help command followed by a command or help section, for example: /help settings Here are the available commands - /start - /whois - /msginfo - /settings - /blacklist, /bl (Operators and agents only) - /log, /l (Operators and agents only) - /prop (Main operator only) - /stats (Main operator only) How do I protect my group from spam? Send /help spam_protection How do I protect my group from unwanted users? Send /help unwanted_users I see false spam being detected in my group Send /help false_detections I think someone got blacklisted incorrectly Send /help false_blacklist I have questions, concerns See @SpamProtectionSupport
